Неупорядоченный список HTML

Ненумерованный список начинается с <ul> тега. Каждый элемент списка начинается с <li> тега.

Элементы списка будут отмечены маркерами (маленькими черными кружками) по умолчанию:

По умолчанию элементы списка будут отмечены цифрами:

square Устанавливает маркер элемента списка в квадрат

Горизонтальный список с помощью CSS

Списки HTML могут быть оформлены различными способами с помощью CSS.

Одним из популярных способов является горизонтальное оформление списка для создания навигационного меню:

<style>

ul {

list-style-type: none;

margin: 0;

padding: 0;

overflow: hidden;

background-color: #333333;

}


li {

float: left;

}


li a {

display: block;

color: white;

text-align: center;

padding: 16px;

text-decoration: none;

}


li a:hover {

background-color: #111111;

}

</style>

Тег HTML <ol> определяет упорядоченный список. Упорядоченный список может быть числовым или алфавитным.

Упорядоченный список HTML

Упорядоченный список начинается с <ol> тега. Каждый элемент списка начинается с <li> тега.

По умолчанию элементы списка будут отмечены цифрами:

  1. Кофе
  2. Чай
  3. Молоко

Упорядоченный список HTML — атрибут Type

Атрибут type <ol> тега определяет тип маркера элемента списка:

Значение Описание
type="1" Элементы списка будут пронумерованы цифрами (по умолчанию)
type="A" Элементы списка будут пронумерованы заглавными буквами
type="a" Элементы списка будут пронумерованы строчными буквами
type="I" Элементы списка будут нумероваться прописными римскими цифрами.
type="i" Элементы списка будут пронумерованы строчными римскими цифрами.

  2. Чай
  3. Молоко

Блок HTML и встроенные элементы

Каждый элемент HTML имеет отображаемое значение по умолчанию, в зависимости от типа элемента.

Есть два отображаемых значения: блочное и встроенное.

Элементы уровня блока

Блочный элемент всегда начинается с новой строки, и браузеры автоматически добавляют некоторое пространство (поле) до и после элемента.

Блочный элемент всегда занимает всю доступную ширину (растягивается влево и вправо, насколько это возможно).

Двумя часто используемыми блочными элементами являются: <p> и <div>.

Элемент <p> определяет абзац в документе HTML.

Элемент <div> определяет раздел или раздел в HTML-документе.

Элемент <p> является элементом уровня блока.

Элемент <div> является элементом блочного уровня.
