HTML Фавикон

Фавикон — это небольшое изображение, отображаемое рядом с заголовком страницы во вкладке браузера.

Как добавить фавикон в HTML

Вы можете использовать любое понравившееся вам изображение в качестве фавиконки.

Совет: фавикон — это небольшое изображение, поэтому оно должно быть простым и контрастным

Изображение фавиконки отображается слева от заголовка страницы на вкладке браузера.

Чтобы добавить значок на свой веб-сайт, либо сохраните изображение значка в корневом каталоге вашего веб-сервера, либо создайте папку в корневом каталоге с именем images и сохраните изображение значка в этой папке. Распространенное название изображения фавикона — «favicon.ico».

Затем добавьте <link> элемент в файл index.html после <title> элемента, например:

<!DOCTYPE html>

<html>

<head>

<title></title>

<link rel="icon" href="img/......">

</head>

<body>


<h1>This is a Heading</h1>

<p>This is a paragraph.</p>


</body>

</html>

Теперь сохраните файл index.html и перезагрузите его в браузере. Теперь на вкладке вашего браузера слева от заголовка страницы должно отображаться изображение вашего фавикона.

HTML-таблицы

Таблицы HTML позволяют веб-разработчикам упорядочивать данные в строки и столбцы.

Определить HTML-таблицу

Таблица в HTML состоит из ячеек таблицы внутри строк и столбцов.

Простая HTML-таблица:

Компания Контакт Страна
Альфред Футтеркисте Мария Андерс Германия
Торговый центр Моктесума Франсиско Чанг Мексика

Ячейки таблицы

Каждая ячейка таблицы определяется тегом a <td> и </td>.

td обозначает табличные данные.

Все, что находится между <td> и </td>, является содержимым ячейки таблицы.

Эмиль Тобиас Линус

Примечание. Ячейка таблицы может содержать все виды HTML-элементов: текст, изображения, списки, ссылки, другие таблицы и т. д.

Строки таблицы

Каждая строка таблицы начинается с <tr> и заканчивается </tr> тегом.

tr обозначает строку таблицы.

Эмиль Тобиас Линус
16 14 10

В таблице может быть сколько угодно строк; просто убедитесь, что количество ячеек одинаково в каждой строке.

Примечание.Бывают случаи, когда в одной строке может быть меньше или больше ячеек, чем в другой. Вы узнаете об этом в следующей главе.

Заголовки таблиц

Иногда вы хотите, чтобы ваши ячейки были ячейками заголовка таблицы. В этих случаях используйте <th> тег вместо <td>тега:

th обозначает заголовок таблицы.

Пусть первая строка будет ячейками заголовка таблицы:

Человек 1 Человек 2 Человек 3
Эмиль Тобиас Линус
16 14 10

По умолчанию текст в <th> элементах выделен жирным шрифтом и расположен по центру, но вы можете изменить это с помощью CSS.

Границы таблицы HTML

Таблицы HTML могут иметь границы разных стилей и форм.

Как добавить границу

Когда вы добавляете границу к таблице, вы также добавляете границы вокруг каждой ячейки таблицы:

Человек 1 Человек 2 Человек 3
Эмиль Тобиас Линус
16 14 10

Чтобы добавить границу, используйте border свойство CSS для элементов table, th и td:

table, th, td {

border: 1px solid black;

}

Свернутые границы таблицы

Чтобы избежать двойных границ, как в приведенном выше примере, установите для border-collapse свойства CSS значение collapse.

Это заставит границы рухнуть в единую границу:

Человек 1 Человек 2 Человек 3
Эмиль Тобиас Линус
16 14 10

table, th, td {

border: 1px solid black;

border-collapse: collapse;

}

Невидимые границы

Если вы установите цвет фона для каждой ячейки и зададите границе белый цвет (такой же, как фон документа), вы получите впечатление невидимой границы:

Человек 1 Человек 2 Человек 3
Эмиль Тобиас Линус
16 14 10

table, th, td {

border: 1px solid white;

border-collapse: collapse;

}

th, td {

background-color: #96D4D4;

}

Круглые границы

Со border-radius свойством границы получают закругленные углы:

Человек 1 Человек 2 Человек 3
Эмиль Тобиас Линус
16 14 10

table, th, td {

border: 1px solid black;

border-radius: 10px;

}

Пропустите границу вокруг таблицы, исключив ее table из селектора css:

Человек 1 Человек 2 Человек 3
Эмиль Тобиас Линус
16 14 10

th, td {

border: 1px solid black;

border-radius: 10px;

}

Пунктирные границы таблицы

С помощью border-style свойства вы можете установить внешний вид границы.

Человек 1 Человек 2 Человек 3
Эмиль Тобиас Линус
16 14 10

Допускаются следующие значения:

  • dotted
  • dashed
  • solid
  • double
  • groove
  • ridge
  • inset
  • outset
  • none
  • hidden

th, td {

border-style: dotted;

}

Цвет границы

С помощью border-color свойства вы можете установить цвет границы.

th, td {

border-color: #96D4D4;

}

Размеры HTML-таблиц

HTML-таблицы могут иметь разные размеры для каждого столбца, строки или всей таблицы.

Используйте style атрибут со свойствами width или height, чтобы указать размер таблицы, строки или столбца.

Ширина таблицы HTML

Чтобы установить ширину таблицы, добавьте к элементу <table> атрибут style:

Установите ширину таблицы на 100%:

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94

Примечание. Использование процента в качестве единицы измерения ширины означает, насколько широким будет этот элемент по сравнению с его родительским элементом, которым в данном случае является <body> элемент.

Ширина столбца таблицы HTML

Чтобы установить размер определенного столбца, добавьте style атрибут к элементу <th> или <td>:

Установите ширину первого столбца на 70%:

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94

Высота строки таблицы HTML

Чтобы установить высоту определенной строки, добавьте style атрибут к элементу строки таблицы:

Установите высоту второй строки на 200 пикселей:

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94

Заголовки таблицы HTML

Таблицы HTML могут иметь заголовки для каждого столбца или строки или для многих столбцов/строк.

Заголовки таблицы HTML

Заголовки таблиц определяются с помощью th элементов. Каждый th элемент представляет ячейку таблицы.

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94

Вертикальные заголовки таблиц

тобы использовать первый столбец в качестве заголовков таблицы, определите первую ячейку в каждой строке как th элемент:

Firstname Jill Eve
Lastname Smith Jackson
Age 94 50

Выравнивание заголовков таблицы

По умолчанию заголовки таблиц выделены жирным шрифтом и расположены по центру:

Чтобы выровнять заголовки таблиц по левому краю, используйте text-align свойство CSS:

th {

text-align: left;

}

Заголовок для нескольких столбцов

У вас может быть заголовок, который охватывает два или более столбца.

Для этого используйте colspan атрибут <th> элемента:

Name Age
Jill Smith 50
Eve Jackson 94

Заголовок таблицы

Вы можете добавить заголовок, который будет служить заголовком для всей таблицы.

Чтобы добавить подпись к таблице, используйте <caption> тег:

Monthly savings
Month Savings
January $100
February $50

Примечание. Тег <caption> должен быть вставлен сразу после <table> тега.

Padding & Spacing HTML-таблицы

Таблицы HTML могут регулировать отступы внутри ячеек, а также пространство между ячейками.

HTML-таблица — Padding

Padding (заполнение ячейки) — это пространство между краями ячейки и содержимым ячейки.

По умолчанию отступ установлен на 0.

Чтобы добавить отступы к ячейкам таблицы, используйте padding свойство CSS:

th, td {

padding: 15px;

}

Чтобы добавить отступ только над содержимым, используйте padding-top свойство.

А остальные стороны со свойствами padding-bottom, padding-left и padding-right:

th, td {

padding-top: 10px;

padding-bottom: 20px;

padding-left: 30px;

padding-right: 40px;

}

HTML-таблица — Spacing

Spacing (Расстояние между ячейками) — это расстояние между каждой ячейкой.

По умолчанию пространство установлено на 2 пикселя.

Чтобы изменить расстояние между ячейками таблицы, используйте border-spacing свойство CSS table элемента:

table {

border-spacing: 30px;

}

Таблица HTML Colspan и Rowspan

Таблицы HTML могут иметь ячейки, которые охватывают несколько строк и/или столбцов.

HTML-таблица — Colspan

Чтобы сделать ячейку, охватывающую несколько столбцов, используйте colspan атрибут:

Name Age
Jill Smith 43
Eve Jackson 57

Примечание. Значение colspan атрибута представляет собой количество столбцов, которые нужно охватить.

HTML-таблица — диапазон строк

Чтобы сделать ячейку, охватывающую несколько строк, используйте rowspan атрибут:

Name Jill
Phone 555-1234
555-8745

Примечание. Значение rowspan атрибута представляет собой количество строк, которые нужно охватить.

Стили HTML-таблиц

Используйте CSS, чтобы ваши таблицы выглядели лучше.

HTML-таблица — полосы зебры

Если вы добавите цвет фона в каждую вторую строку таблицы, вы получите приятный эффект полос зебры.

Чтобы стилизовать любой другой элемент строки таблицы, используйте :nth-child(even) селектор следующим образом:

tr:nth-child(even) {

background-color: #D6EEEE;

}

Примечание. Если вы используете (odd) вместо (even), стиль будет применен к строкам 1, 3, 5 и т. д., а не к 2, 4, 6 и т. д.

HTML-таблица — вертикальные полосы зебры

Чтобы сделать вертикальные полоски зебры, стилизуйте каждый второй столбец, а не каждую вторую строку.

Установите :nth-child(even) для элементов данных таблицы следующим образом:

td:nth-child(even), th:nth-child(even) {

background-color: #D6EEEE;

}

Примечание.Поместите :nth-child() селектор на оба элемента th и td , если вы хотите, чтобы стиль применялся как к заголовкам, так и к обычным ячейкам таблицы.

Комбинируйте вертикальные и горизонтальные полосы зебры

Вы можете комбинировать стиль из двух приведенных выше примеров, и у вас будут полосы в каждой второй строке и каждом втором столбце.

Если вы используете прозрачный цвет, вы получите эффект перекрытия.

Используйте rgba( ) цвет, чтобы указать прозрачность цвета:

tr:nth-child(even) {

background-color: rgba(150, 212, 212, 0.4);

}


th:nth-child(even),td:nth-child(even) {

background-color: rgba(150, 212, 212, 0.4);

}

Горизонтальные разделители

Если вы укажете границы только внизу каждой строки таблицы, у вас будет таблица с горизонтальными разделителями.

Добавьте border-bottom свойство ко всем tr элементам, чтобы получить горизонтальные разделители:

tr {

border-bottom: 1px solid #ddd;

}

Парящий стол

Используйте :hover селектор tr, чтобы выделить строки таблицы при наведении указателя мыши:

tr:hover {background-color: #D6EEEE;}
